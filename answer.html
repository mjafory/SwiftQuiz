Quiz Explanations

1. What is the result of the expression x + y * z?
if x = 2,
y = 3,
z = 5?
Correct Answer:B) 17
Explanation: The expression follows the order of operations (BIDMAS/BODMAS). First, the multiplication is performed (3 * 5 = 15), and then the addition (2 + 15), resulting in 17.

2. Given
var firstNumber = 3.0
var secondNumber = 4.4
what is the result of firstNumber + secondNumber?
Correct Answer: A) 7.4
Explanation: The addition of firstNumber (3.0) and secondNumber (4.4) results in 7.4.

3. Suppose you're checking the weather with two conditions:
var isSnowing = false
var isRaining = true
var isWarm = isSnowing || isRaining
 Correct Answer: A) true
Explanation: The || (logical OR) operator returns true if at least one of the conditions is true. In this case, isRaining is true, so isWarm is true.

4. Which escape character is used for a new line in Swift strings?
Correct Answer: D) \n
Explanation: The \n inside the string represents a new line.

5. What is the output of the following code?
let age = 55
if age > 50 {
  print("a is less than 20")
} else {
  print("a is greater than 20")
}
 Correct Answer: B) a is greater than 20
Explanation: The condition age > 50 is true, so the code inside the else block is executed.
